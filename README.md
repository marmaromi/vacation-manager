# vacation-manager

1. users can track their favourite vacations
2. admins can add, update and delete vactions and view statistics of the followed vacations.
3. the site updates user/admin changes in real time with Socket.IO

## built with:
- Database: 
  - MySQL
- Backend: 
  - Typescript
  - Node.js
  - Express
  - Socket.IO
- Frontend: 
  - React + TypeScript
  - Redux
  - CSS
